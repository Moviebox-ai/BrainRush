# BrainRush

BrainRush is a Kotlin + Jetpack Compose Android puzzle-game starter built around MVVM and Clean Architecture. The scaffold includes an offline-first repository, Room persistence, Hilt dependency injection, Material 3 screens, Navigation Compose, Firebase-ready dependencies, AdMob-ready abstractions, and production-oriented folders for future puzzle engines and live-ops features.

## Highlights

- Package: `com.aaryo.brainrush`
- Minimum SDK: 24
- Compile/Target SDK: 35
- UI: Jetpack Compose + Material Design 3
- Architecture: MVVM + Clean Architecture
- Offline first: Room-backed repository with Flow state streams
- Ready integrations: Firebase Auth, Firestore, Remote Config, Crashlytics, Analytics, FCM, and Google Mobile Ads dependencies

## Build

Use Android Studio or AndroidIDE with a configured Android SDK, then run:

```bash
gradle :app:assembleDebug
```
